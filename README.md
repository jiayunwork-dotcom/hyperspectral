# 高光谱遥感影像分类与地物识别系统

基于Streamlit的高光谱遥感影像分类与地物识别完整流程应用。

## 功能特性

### 📥 数据导入
- 支持ENVI格式的高光谱数据（BSQ/BIL/BIP三种交叉存储格式）
- 同时上传 `.hdr` 头文件和对应的二进制数据文件
- 自动解析头文件获取行列数、波段数、数据类型、波长列表等元信息
- 采用内存映射（memory map）方式加载大文件，支持几百MB级别数据
- 侧边栏实时显示影像基本信息（尺寸、波段数、数据类型、坐标信息）

### ⚙️ 数据预处理
**噪声波段剔除**
- 用户手动指定波段序号范围
- 系统自动检测信噪比（SNR）低于阈值的波段并移除

**MNF最小噪声分离变换**
- 相邻像素差分法估计噪声协方差矩阵
- 噪声白化处理后PCA提取信号分量
- 用户可选择保留前N个MNF分量
- 大影像采用随机采样子集估计协方差矩阵

**PCA降维**
- 直接对原始数据做主成分分析
- 累积贡献率曲线辅助用户决定保留分量数
- 支持方差阈值自动选择分量数

**光谱平滑**
- Savitzky-Golay滤波
- 窗口大小和多项式阶数可调
- 支持逐行/分块处理大文件

### 🔬 特征提取
**光谱特征**
- 连续统去除（包络线拟合后归一化）
- 一阶/二阶光谱导数
- 特定吸收峰的位置和深度自动检测

**空间特征**
- 形态学剖面（多尺度开闭运算提取结构信息）
- Gabor纹理特征（多方向多频率）

**特征融合**
- 支持光谱特征、空间特征或两者融合作为分类输入

### 🤖 分类算法
**SVM（RBF核）**
- 网格搜索自动优化C和gamma参数
- 交叉验证选择最优参数

**随机森林**
- 可调树数量、最大深度、特征采样比例
- 输出特征重要性

**1D-CNN**
- 将每个像素的光谱向量作为一维序列输入
- 网络结构：3层卷积 + 全连接层

**3D-CNN**
- 取每个像素周围空间邻域窗口（如7x7）
- 联合光谱维度形成三维数据立方体输入
- 分块推理策略消除边界伪影

**半监督学习**
- 标签传播算法
- 利用少量标注样本和大量无标注像素的空间邻域关系

### 📝 训练样本管理
- 导入地面真实标签文件（与影像同尺寸的分类标注图，0表示未标注）
- 交互式ROI选取：矩形/多边形工具圈选感兴趣区域并指定类别标签
- 样本管理面板显示各类别样本数量、空间分布、类别间样本均衡度
- 类别不均衡时提供SMOTE过采样选项
- 训练集/测试集划分（默认7:3），支持分层采样保证各类别比例

### 📊 精度评估
- 总体精度（OA）、平均精度（AA）、Kappa系数
- 每个类别的精确率/召回率/F1分数
- 混淆矩阵以热力图形式展示
- 支持归一化和原始计数两种模式切换

### 🖼️ 分类结果可视化
- 彩色编码的地物分类图，每个类别用不同颜色填充
- 图例标注类别名称和颜色对应关系
- 支持分类图半透明叠加在原始影像上
- 支持单类提取：只显示某一特定地物类别的分布区域
- 波段组合工具：真彩色/标准假彩色/NDVI指数图/自定义三波段合成

### 📦 批量处理
- 支持同时上传多景影像
- 使用相同的分类参数和训练样本对多景影像批量执行分类
- 结果逐一保存为ENVI格式和PNG图像
- 分块推理策略：将大影像切分为重叠块，分块预测后拼接消除块边界伪影

### 🚀 性能优化
- 超过500MB的数据文件采用逐行或分块处理
- MNF变换对大影像用随机采样子集估计协方差矩阵
- 3D-CNN训练时样本数超过10000则使用mini-batch（batch ≤ 256）
- 内存映射技术避免一次性加载全部数据到内存

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
streamlit run app.py
```

## 系统要求

- Python 3.8+
- 推荐内存：8GB+
- 可选：CUDA GPU（用于加速深度学习模型训练）

## 使用流程

1. **数据导入**：上传ENVI格式的高光谱数据（.hdr + 数据文件）
2. **数据预处理**：选择需要的预处理步骤并执行
3. **特征提取**：选择光谱/空间特征类型并提取
4. **样本管理**：导入标签或手动选取ROI，划分训练/测试集
5. **模型训练**：选择分类算法，设置参数，训练模型
6. **精度评估**：计算评估指标，查看混淆矩阵
7. **分类可视化**：查看分类结果，支持多种显示模式和波段组合
8. **批量处理**（可选）：上传多景影像，批量分类并保存结果

## 支持的数据格式

- ENVI标准格式（.hdr头文件 + 二进制数据文件）
- 支持BSQ、BIL、BIP三种存储格式
- 支持的数据类型：uint8, int16, int32, float32, float64, uint16, uint32, int64, uint64
